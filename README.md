# 🚀 Space Travel

A React-based single-page application that simulates interplanetary transportation and population management across the solar system. Users can build spacecraft, manage planetary populations, and transfer people between planets through an interactive interface.

## Features

- Create and manage spacecraft
- View detailed spacecraft information
- Remove retired spacecraft
- Monitor planetary populations
- Transfer passengers between planets
- Dynamic updates using React state management
- Client-side routing with React Router
- Loading state management using React Context
- Persistent data storage with Local Storage

## Technologies Used

- React
- React Router
- JavaScript (ES6+)
- Context API
- CSS Modules
- Local Storage

## Key Concepts Demonstrated

- Component-based architecture
- State management
- Context API
- CRUD operations
- Routing and navigation
- Form validation
- Conditional rendering
- Data persistence
